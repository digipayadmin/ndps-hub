NDPS: Digital Architecture Hub

The Master Infrastructure Link-in-Bio & Corporate Calling Card for New Digital Payment Systems (NDPS) LLC.
Overview: This repository houses the digital entry point for the NDPS ecosystem. It serves as a unified routing hub, 
directing distinct verticals (Enterprise B2B, Retail, Live Events, and Artists) to their specific software 
platforms and orchestration environments.NDPS operates on a strict Data Layer Orchestration Model, utilizing 
NMI as our primary switchboard to dynamically bifurcate transactions between fiat settlement rails (Priority, 
PayRoc) and digital asset rails (Nuvei, BitPay).

Ecosystem Topology: 
(The "Tri-Rail" Framework) This hub connects visitors to the three core nodes of our architecture:
Node 1: OmniPOS (omni-pos.io) The Front-End: The merchant-facing UI, virtual terminal, and Smart Split dashboard. 
Focused on UX, margin recovery, and checkout conversion.
Node 2: POSitiveRails™ (positiverails.com) The Back-End Engine: The developer portal and proprietary routing logic. 
This invisible layer sits inside NMI, automating Level 3 data injection and enforcing RewardPay compliance.
Node 3: NDPS Corporate (This Repository) The Master Entity: The institutional parent holding company, 
managing banking relationships, ISO contracts, and API Master Keys.

Vertical Destinations
The ndps_hub.html file acts as the primary traffic controller for:
OmniPOS Live: Smart contract royalty splits for touring artists and musicians.
Festival & Venue Operations: Automated cross-revenue F&B routing and merch cuts.
Retail & Boutique Hospitality: Zero-fee surcharging via PayRoc.Commercial B2B: High-ticket freight and 
logistics interchange optimization via Priority.
